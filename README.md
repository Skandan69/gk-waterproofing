# G.K Waterproofing & Painting Works

Responsive static business website built with HTML, CSS and minimal JavaScript.

## Preview locally

Open `index.html` in a browser. For the most accurate preview, serve this folder with any simple static web server.

## Deploy to Cloudflare Pages

1. Push this repository to GitHub.
2. In Cloudflare, open **Workers & Pages** and choose **Create application → Pages → Connect to Git**.
3. Select the `gk-waterproofing` repository.
4. Choose **None** as the framework preset, leave the build command empty, and use `/` as the output directory.
5. Save and deploy.

The website deploys directly from the repository root. No build process or environment variables are required.
